ZinJS.js
============
www.zinjs.com

We want to bring new dimension to internet presentations. 
Our framework is based on CSS3 and Javascript.

WHY ZinJS?
----------
ZinJS = Zoom IN JavaScript or Zoom Is Not Just a Script
